# getLocalNetatmoWeather
Parse Address and get local Weather from Netatmo

This script parse the Address to geo coordinates and get the local wether from Netatmo

getWeather.php?adress=Chicago&distance=2

Parameters:  
* address ...  
* distance (optionl default="1"km) ... Weather station search radius  
* value (optionl default="temperatur") ... switch between "temperature" and "humidity" (all Information Public NetAtmo Api shows  
* debug (optional ="false") shows verbose informations
